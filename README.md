# KCM — Расчёт центра масс твёрдого тела / Center of Mass Calculation

[Русский](#русский) | [English](#english)

---

## Русский

Программа для определения координат центра масс по реакциям в опорах.

- Учёт трения в шарнирах
- Расчёт неисключённой систематической погрешности (НСП) по ГОСТ
- 3D-визуализация
- Сохранение/загрузка проектов

### Установка
Скачайте KCM.exe из Releases или запустите `python main_window.py`

### Требования
Python 3.10+, PySide6, NumPy, Matplotlib, numpy-stl

---

## English

A tool for determining the center of mass coordinates from support reactions.

- Friction in joints taken into account
- Estimation of non-eliminated systematic uncertainty (according to GOST)
- 3D visualization
- Project save/load

### Installation
Download KCM.exe from Releases or run `python main_window.py`

### Requirements
Python 3.10+, PySide6, NumPy, Matplotlib, numpy-stl
